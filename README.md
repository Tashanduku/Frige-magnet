# Fridge Magnet

A simple interactive app that lets users create, move, and arrange digital “fridge magnet” words in real time.  
Built with Flask, Socket.IO, and a lightweight frontend.

---

## Features
- Drag-and-drop word magnets  
- Real-time syncing across clients via WebSockets  
- Server-stored positions for persistence  
- Clean, minimal interface  

---

## Tech Stack
**Backend:** Flask, Flask-CORS, Flask-SocketIO  
**Frontend:** HTML, CSS, JavaScript  
**Storage:** JSON file  

---
